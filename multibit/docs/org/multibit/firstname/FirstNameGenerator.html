Class FirstNameGenerator

java.lang.Object
  org.multibit.firstname.FirstNameGenerator

public class FirstNameGenerator
extends java.lang.Object

class to generate firstnames from a target string firstnames are of the format: Jim Burton.lk2 - first NUMBER_OF_CHARACTERS_TO_MATCH_IN_NAME chars are used The first bits address is created with the first characters of the target string appended with the characters after the final dot thus the first bits from this first name is '1jimlk2' The target string (e.g. "Jim Burton") is lower cased and illegal chars mapped to their legal equivalents
